The Capital Code — Architect Edition
5 interactive wealth-building playbooks for serious investors.
The Capital Code is a complete, self-contained investment system built for private investors who want to think and operate at institutional level. Each playbook covers a distinct pillar of wealth management — from strategy and crisis response to tax optimization, case study analysis, and operational templates. All tools run fully interactive in the browser, no apps or installations required.

How to Use
Open index.html or navigate to the GitHub Pages link to access all five playbooks. Each playbook is a standalone interactive web application — no internet connection required once loaded, no data is transmitted, all inputs are processed locally in your browser.
